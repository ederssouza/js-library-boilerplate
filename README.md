# JavaScript Library Boilerplate

## Dependencies versions

- **@babel/cli:** 7.12.1
- **chai:** 4.2.0
- **clean-webpack-plugin:** 3.0.0
- **eslint:** 7.12.1
- **mocha:** 8.2.0
- **webpack:** 5.3.0
- **webpack-cli:** 4.1.0

## package.json scripts

### Tests

```bash
# run unit tests
$ yarn test

# run unit tests on watch mode
$ yarn test:watch

# run unit and coverage tests
$ yarn test:coverage
```

### Build

```bash
# run the production build
$ yarn build
```
